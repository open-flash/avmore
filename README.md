# Avmore

Open Flash Actionscript Virtual Machine

## Get traces

```bash
echo "ErrorReportingEnable=1" > ~/mm.cfg
flashplayerdebugger myfile.swf
cat ~/.macromedia/Flash_Player/Logs/flashlog.txt
```
